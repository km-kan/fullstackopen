# 0.6: New note in Single page app diagram

```mermaid
sequenceDiagram
    participant browser
    participant server

    browser->>server: POST https://studies.cs.helsinki.fi/exampleapp/new_note_spa
    activate server
    server->>browser: Confirmation message of new note successfully being added 

    Note right of browser: The new note is added to a notes list, the HTML page is rerendered with <br> the new note, and the note is sent to the server. No redirection occurs.
```
